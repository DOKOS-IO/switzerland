Localization Application for Switzerland

This application is designed to add localization options for Switzerland to ERPNext

Install

From your site

To install this app, login to your site and click on "Installer". Search for Switzerland and click on "Install"

Using Bench

Go to your bench folder and setup the new app

$ bench get-app switzerland https://github.com/DOKOS-IO/switzerland
$ bench new-site testsite
$ bench --site testsite install-app switzerland

Login to your site to configure the app.
